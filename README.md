Create at least 3 files
- Letter.js
  - Does not import any other files
  - Constructor for letters
- Word.js
  - Imports from Letter
   
- Index.js
  - Imports from Letters
  - Prompts user to guess letter
  - Randomly selects words
  - Game Logic
